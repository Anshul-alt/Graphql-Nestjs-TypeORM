## Description

This is a starter repository for building a NestJS application with GraphQL and TypeORM. This project provides a robust setup for developing scalable server-side applications using GraphQL as the API language and TypeORM as the ORM for managing database interactions.

## Features

- **NestJS**: A progressive Node.js framework for building efficient and scalable server-side applications.
- **GraphQL**: Integrated GraphQL module to enable flexible data querying.
- **TypeORM**: Powerful ORM that supports multiple database systems, including PostgreSQL, MySQL, SQLite, and more.
- **TypeScript**: Typed JavaScript at any scale.
